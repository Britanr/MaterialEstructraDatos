Fundamentos e Introducción

Bases conceptuales de estructuras de datos

Resultado de aprendizaje: Identifica las características y fundamentos de las estructuras de datos mediante el análisis de bibliografía especializada y casos prácticos.

Temas principales

  • Introducción: Historia, Características, Ventajas y Desventajas
  • Tipos de Datos: Primitivos, Compuestos, Agregados y Abstractos
  • Datos compuestos: Vectores, Matrices, Registros
  • Fundamentos de Algoritmia: Concepto y Representación de algoritmos

Estructuras Lineales

Listas, pilas y colas

Resultado de aprendizaje: Analiza y aplica estructuras de datos lineales (listas, pilas y colas) para resolver problemas computacionales.

Búsqueda y Ordenamiento

Algoritmos fundamentales

Resultado de aprendizaje: Conoce y aplica los métodos comunes de ordenamiento y búsqueda para cada estructura de datos estudiada.

Temas principales

  • Análisis de algoritmos: Complejidad temporal y espacial (Big O)
